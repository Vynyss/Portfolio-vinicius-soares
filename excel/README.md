# Projetos em Excel

Planilhas avançadas com foco em automação de relatórios e visualização de dados para tomada de decisão.

🔧 Funcionalidades usadas:
- Tabelas dinâmicas
- Validação de dados
- Fórmulas condicionais (SE, PROCV, ÍNDICE, CORRESP)
- Dashboards simples com gráficos interativos

📁 Cada projeto contém:
- Arquivo `.xlsx`
- Prints das planilhas e relatórios
- Explicação do objetivo e insights gerados
